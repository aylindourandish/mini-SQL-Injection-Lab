# Mini SQL Injection Lab

This project demonstrates SQL Injection vulnerabilities using a simple
login simulation.

It includes:
- A vulnerable login implementation
- A secure login implementation
- Detection of common SQL Injection patterns

The project is intended for educational and security learning purposes.

## Features
- Demonstrates authentication bypass attacks
- Compares insecure vs secure logic
- Runs in a loop for repeated testing
- Written in C++ / Python for learning purposes

## Disclaimer
This project is for educational use only.
